Family Glucose Log

Home page and technical support page for App Store submission.

Family Glucose Log is a privacy-first iOS app for recording blood glucose data for multiple family members.

  • Fast glucose logging with tags and notes
  • Member-based history and trend charts
  • PDF export for clinic visits
  • Local storage with optional iCloud private sync

Technical Support

Email: support@glucoselog-app.newtree.tech

Support hours: Monday to Friday, 09:00-18:00 (UTC+8)

Typical response time: within 2 business days.

FAQ

Does this app provide medical advice?
No. It only logs and visualizes data.

How is data synced?
Data stays local by default and can sync via iCloud private database.
